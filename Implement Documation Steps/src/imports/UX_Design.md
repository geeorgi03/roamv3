# UX Design Reference

## What UX Should Look Like — Use Cases to Solve

---

## MirrorTube Extension

**Features:** Loop, Mirror, Speed control, One loop at a time, Repeat

A browser extension for YouTube that adds a side panel with:
- **Mirror** toggle checkbox
- **Speed** slider (e.g. x1.0)
- **Repeat** — enable with start/end time inputs and manual input option

---

## CapCut

**Features:** Cropping, editing, trimming, scale, changing audio, videos

A video editing desktop application with:
- Left panel: media library (device, presets, spaces, brand assets)
- Center: video preview player with timeline scrubber
- Right panel: Video / Audio / Speed / Animation / Adjustment tabs
  - Transform controls: Scale, Position (X/Y), Rotate
  - Uniform scale toggle
  - Blend options
  - Stabilize (Pro)
- Bottom: multi-track timeline editor

---

## Choreography Position Mapping — YouTube Reference

**"Girls Never Die - TripleS placements guide with mirrored dance practice"**

A YouTube video overlaying a grid on top of the dance practice footage, with labeled dots showing each member's stage position (numbered 1–24 with names).

Indicators for members leaving stage left (→) or right (←).

---

## ArrangeUs — 2D/3D Formation Tool

**Use case:** Choreograph and map dancer positions in formations.

**Limitation:** Not precise — can only pinpoint static locations, not capture full movement. Position mapping is only one part of choreography.

**Features:**
- 2D and 3D views (3D available on Android)
- Stage grid with BACKSTAGE / AUDIENCE orientation
- Drag-and-drop dancer dots with name labels, color-coded by group/role
- Formation tabs (e.g. Formation 57, Formation 58) + "Add formation" button
- Undo/redo, save, playback controls

---

## STEEZY — Notetracks

**Use case:** Listening to music to take notes, sketch drawings, add symbols.

**Features:**
- Waveform audio timeline with playhead scrubber
- Timestamp markers on the waveform
- Lyrics panel overlay
- Track import: iTunes, Dropbox, Wifi transfer, Audio Paste
- Note types: Text labels, Symbols, Drawings — placed directly on the timeline
- Example label tags: "Stab synth rising", "Full Drums", "Kick only", "Low end bass", "Bass riff arp"

> *"Notetracks lets you take notes, sketch drawings, and add symbols along the song."*
